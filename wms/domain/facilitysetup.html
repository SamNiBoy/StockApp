Facility Setup

When we considering setup a warehouse, there are certain aspects:

  • Dock & Stage
  • Storage Zone
  • Picking Zone
  • Movement Zone
  • Pick and Deposit

Dock Location: Dock is a location, it defines a physical dock slot either used by receiving truck or by shipping truck.

Stage Location: Stage is a location, it defines a physical location used for storing pallets before Putaway(Receiving Stage) or before Loading trailer(Shipping Stage). for shipping stage location, system can be setup with a resource code that can be a Shipment ID, a Stop ID, or a Load, this way system will assure only inventory belongs to the same Shipment/Stop/Load can be stored in the same stage location; it can also setup with capacity tracked if the stage location is space critical.

Storage Zone: Defines a group of locations that can be used for inventory Putaway, typically Storage Zone can be added to Storage Configuration Rule to tell system storage preference during Putaway system allocate a location.

Pick Zone: Defines a set of locations that system can use it to configure outbound picking. e.g we can defined pallet pick zone for pallet pick, case picking zone for case pick and each picking zone for detail pick.

Movement Zone: Defines some features when inventory move in/out of this zone, a typical user case is when receiving carton, cartons needs to be built into pallet before Putaway, in this case we can define a Movement Path that when move inventory from Receive Stage Movement Zone to Storage Movement Zone, go to Hop Movement Zone 'Pallet Build' zone; For outbound scenario, Hop area is more often used for outbound picked inventory to provide additional process, for example for each carton picked, need to go to movement zone for labeling service.

Pick and Deposit: PnD location is a special location where system allows you deposit inventory into the location and then pickup from the location as needed. e.g, when do List Picking, user can 'set down' the half-way picked list into a PnD location, and 2nd operator can resume the list pick by scanning the LPN from PnD location.
