Trace Digger

Trace Digger was developed in 2010, at that time I worked in WMS maintainance team, in daily basis I need to investigate a lot of traces for troubleshooting, we already have a very good tool named LextEdit for investigating MOCA structed trace, however, given the frequency of investigating trace, I need a tool which can give me more flexibilty:

  • Ability for Quick Search and Navigating
  • Order Command/SQL by Performance
  • Bookmark a Command/SQL for Switching between Nodes
  • Ability for Investigating MTF trace
  • ...

There is a document on the internet talked about more detail, See Here for more.

The latest version is 2.1, comparing to previous version, a few issues fixed:

  • Used a new font for hilighted trace
  • Made the search bar more wider, so can type more chars to search
  • Fixed the issue of can not open Unix system generated trace file
